# Igreja Saudável

Aplicativo para diagnóstico e revitalização de igrejas em declínio.

## Funcionalidades
- Cadastro da igreja
- Questionário com 18 perguntas (6 áreas)
- Dashboard com notas e gráfico
- Plano de ação personalizado
- Histórico de diagnósticos
- Exportação de relatório

## Tecnologias
- Flutter (UI)
- Firebase Firestore (banco de dados)
- fl_chart (gráficos)

## Como configurar
1. Clone o repositório
2. Crie um projeto no Firebase
3. Ative o Firestore
4. Importe as perguntas (assets/perguntas.json)
5. Configure o Firebase no app (flutterfire configure)
6. Rode: flutter pub get && flutter run
